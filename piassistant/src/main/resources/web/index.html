SKILLS · COMPOSIO
Composio provides hundreds of tool integrations (GitHub, Gmail, Slack, Notion, etc.) to OpenClaw via MCP. Paste your consumer key from dashboard.composio.dev, then toggle the toolkits you want exposed to the AI.
REMOTE ACCESS · TAILSCALE FUNNEL
Tailscale Funnel exposes the master device on a permanent public HTTPS URL (e.g. https://your-pi.tail-net.ts.net) without opening router ports. It requires Tailscale installed and logged in on the master. Enable shared-password login below before turning Funnel on — Funnel publishes the app to the public internet.
